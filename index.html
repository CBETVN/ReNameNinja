RenameNinja Plugin Help

Your guide to using the RenameNinja plugin efficiently.

Getting Started

  1. Install the RenameNinja plugin from the official source.
  2. Open your project and activate the plugin.
  3. Access the plugin via the toolbar or context menu.

How to Use

  • Batch Rename: Select files or items, then choose Rename with RenameNinja.
  • Custom Patterns: Use wildcards or regular expressions for advanced renaming.
  • Preview Changes: Review the changes before applying them.

Examples

Pattern: IMG_*
Result: Vacation_*
        
Pattern: file(\d+).txt
Result: document$1.txt

FAQ

  • Can I undo a rename? Yes, use the undo option in the plugin menu.
  • Does it support subfolders? Yes, enable "Include subfolders" in settings.
