ReName Ninja - Help & Features

ReName Ninja is a Photoshop panel plugin designed to make renaming and organizing your layers fast, flexible, and frustration-free.

Main Features

  1. Behavior options
    • Choose whether to apply changes only to selected layers or to all layers in your document.
  2. Batch Rename Layers
    • Rename Modes:
      • Prefix: Add text before the existing layer name.
      • Name: Replace the entire layer name.
      • Suffix: Add text after the existing layer name.
    • Enumerate: Automatically adds a number to each renamed layer for easy ordering. You can enumerate layers either alongside renaming or as a standalone action. To only add numbering to existing layer names, simply leave the name field empty and enable "Enumerate". Numbering follows the order in the Layers panel or in your selection, starting from the top — the topmost layer becomes 01, the next 02, and so on. If the name field is empty and enumeration is not enabled, the plugin will display an alert and cancel the operation. To help avoid accidental numbering, the "Enumerate" checkbox automatically turns off after each run, preventing messy results if left on by mistake.
  3. Find and Replace
    • Find: Enter text to search for in layer names.
    • Replace: Enter replacement text, or use special modes:
      • Delete: Remove all occurrences of the find text.
      • Delete All After: Remove everything after the find text.
      • Delete All Before: Remove everything before the find text.
      • Replace With: Replace found text with your REPLACE text.
      • Insert Before: Insert your REPLACE text before the found text.
      • Insert After: Insert your REPLACE text after the found text.
  4. How the Revert Buttons Work

    ReName Ninja provides two Revert buttons—one for the Rename section and one for the Find and Replace section. However, both buttons perform the same action:
    They restore all your layer names to what they were before your most recent batch rename or find-and-replace operation.

    How It Works

    • Whenever you use either the Rename or Find and Replace Apply button, the plugin saves the current names of all layers.
    • Clicking either Revert button will undo the last batch operation (whether it was a rename or a find-and-replace) and restore all layer names to their previously saved state.
    • After you revert, both Revert buttons are disabled until you perform another batch operation.

    Important Notes

    • One-Level Undo: Only the most recent batch operation can be reverted. If you perform multiple operations, only the last one can be undone.
    • Document-Specific: The revert memory is tied to the current Photoshop document. Switching documents clears the revert state.
    • Visibility: When reverting, only the layer names are restored; other properties like visibility are preserved.
