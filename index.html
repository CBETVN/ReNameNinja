ReName Ninja – Help & Features

ReName Ninja is a Photoshop panel plugin designed to make renaming and organizing your layers fast, flexible, and frustration-free.

Main Features

  1. Batch Rename Layers
    • Apply to Selected / All: Choose whether to rename only selected layers or all layers in your document.
    • Rename Modes:
      • Prefix: Add text before the existing layer name.
      • Name: Replace the entire layer name.
      • Suffix: Add text after the existing layer name.
    • Enumerate: Automatically add a two-digit number to each renamed layer for easy ordering.
  2. Find and Replace
    • Find: Enter text to search for in layer names.
    • Replace: Enter replacement text, or use special modes:
      • Delete: Remove all occurrences of the find text.
      • Delete All After: Remove everything after (and including) the find text.
      • Delete All Before: Remove everything before (and including) the find text.
      • Replace With: Replace found text with your replacement.
      • Insert Before: Insert your text before the found text.
      • Insert After: Insert your text after the found text.
    • Apply to Selected / All: Choose whether to run the operation on selected layers or all layers.
  3. Revert Changes
    • Revert Rename: Undo your last batch rename and restore original layer names.
    • Revert Replace: Undo your last find-and-replace operation.

Usage Tips

  • Input Limits: Layer names, find, and replace fields are limited to 50 characters for stability.
  • Safety Checks: The plugin prevents you from deleting a layer’s name entirely.
  • Smart Objects: Works with all layer types, including Smart Objects.
  • No Selection Warning: If you try to apply changes to selected layers but none are selected, you’ll get a helpful alert.
  • Non-Latin Characters: Alerts you if your find text isn’t found, including a reminder to check for non-Latin characters.

Troubleshooting

  • If you see a message about “no stored names,” perform a rename or replace operation first.
  • If you try to revert more than once, you’ll be notified that there are no more saved revert states.
  • If you exceed input length, you’ll be prompted to shorten your text.

Shortcuts & Best Practices

  • Use Enumerate for quick numbering of layers.
  • Use Apply to All for global changes, or Apply to Selected for targeted edits.
  • Use Revert buttons to quickly undo your last operation if you make a mistake.
